# YouTube-Comments-Sentiment-Analysis
A machine learning project that analyzes YouTube comments and classifies them into Positive, Negative, or Neutral sentiments.

Features
- Reads YouTube comments dataset
- Performs text preprocessing and cleaning
- Generates sentiment labels using VADER Sentiment Analysis
- Balances dataset using resampling
- Converts text into numerical features using CountVectorizer
- Trains a Naive Bayes classification model
- Evaluates model accuracy using confusion matrix and accuracy score

Workflow
- Load comments dataset
- Clean and preprocess text
- Generate sentiment labels
- Balance dataset
- Vectorize text data
- Train Gaussian Naive Bayes model
- Evaluate model performance
